### Running

Prerequisites - Java 8

Compile

```
javac *.java
```

Run

```
java Main <test_file> <window_size>
```

Example

```
java Main ../resources/test2.csv 100
```

If you don't provide any arguments, the program will use example data.




Running .jar (if you have only JRE)

```
java -jar Median.jar
```
